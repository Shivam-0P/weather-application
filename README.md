# 🌦️ Weather Application

A modern and responsive **Weather Application** built using React and Vite that provides real-time weather information for any city worldwide.

---

## 🚀 Overview

This application allows users to search for any location and instantly retrieve current weather data such as temperature, humidity, and conditions. It is designed with a focus on performance, clean UI, and scalability.

---

## ✨ Key Features

* Real-time weather data fetching
* City-based search functionality
* Displays temperature, humidity, and weather conditions
* Responsive and user-friendly interface
* Fast development and build using Vite

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Build Tool:** Vite
* **Styling:** CSS
* **API Integration:** OpenWeather API

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/Shivam-OP/weather-application.git
```

2. Navigate into the project directory:

```
cd weather-application
```

3. Install dependencies:

```
npm install
```

4. Start the development server:

```
npm run dev
```

---

## 🔑 Environment Variables

To run this project, you need to configure your API key.

Create a `.env` file in the root directory and add:

```
VITE_API_KEY=your_api_key_here
```

## 🌐 Usage

1. Enter a city name in the search field
2. Submit the request
3. View real-time weather data instantly
